## 🎯 نظرة عامة

**SchoolLink AI** هو نظام إدارة مدرسية شامل يجمع بين:

- 📚 **إدارة تعليمية كاملة**: متابعة الحضور، الدرجات، الامتحانات، والتقييمات
- 🤖 **ذكاء اصطناعي متقدم**: تحليل الأداء، توليد المحتوى، إنشاء امتحانات ذكية
- 💬 **تواصل فوري**: ربط المدرسة بأولياء الأمور والطلاب
- 📊 **تقارير تلقائية**: تحليلات شهرية وتوصيات مخصصة لكل طالب
- 🎓 **مكتبة رقمية**: رفع وتحميل الكتب والملازم والمذكرات

### 3️⃣ ولي الأمر 👨‍👩‍👧

**الصلاحيات:**
- 📊 متابعة درجات الطالب في جميع المواد
- 📅 استقبال تقارير شهرية تلقائية عبر البريد الإلكتروني / SMS
- 🔔 استقبال إشعارات فورية (ضعف دراسي، غياب، إنذارات سلوكية)
- 💬 التواصل المباشر مع المدرسين والإدارة
- 📖 متابعة الحضور والغياب
- 🎯 الاطلاع على خطط التحسين المقترحة للطالب
- ❓ طرح استفسارات عن التقييمات والغياب والمشاكل
- 🤖 **مساعد ذكي (AI Chatbot)** للإجابة على الأسئلة
- 📋 **التقييم الدراسي الرسمي** (رصد المدرسين الأسبوعي والنتائج النهائية)
- 📱 **التقييم التدريبي الذكي** (أداء الطالب داخل المنصة)

**ربط ولي الأمر بالطالب:**

عند إنشاء حساب ولي الأمر يتم ربطه بالطالب مع تحديد صلة القرابة:

| صلة القرابة | العرض في نظام الشات |

| أب | أحمد علي (والد الطالب محمد أحمد) |
| أم | فاطمة حسن (والدة الطالبة سارة) |
| جد / جدة | يوسف سعيد (جد الطالب عمر) |
| أخ / قريب | حمزة علي (أخ الطالب محمود) |

> 💡 **الفائدة:** يرى المدرسون والإدارة هوية المتحدث كاملةً في الشات مباشرةً دون الحاجة للسؤال عنها.

## ✨ المميزات الأساسية

### 1. 📊 نظام Excel الذكي

**الاستيراد (Import):**
- رفع ملف Excel يحتوي على:
  - أسماء الطلاب
  - الدرجات في جميع المواد
  - الحضور والغياب
  - التقييمات السلوكية

**المعالجة التلقائية:**
- قراءة الملف تلقائيًا
- ربط كل طالب بدرجاته
- تحديث قاعدة البيانات
- حساب المتوسطات والمؤشرات

**التصدير (Export):**
- **تقارير PDF مخصصة** لكل طالب
- **ملفات Excel ملخصة** للفصل أو المدرسة
- **رسوم بيانية** توضح الأداء

**مثال على ملف Excel المدخل:**
```
| اسم الطالب    | الرياضيات | العربي | الإنجليزي | العلوم | الحضور |
|---------------|-----------|--------|-----------|--------|--------|
| أحمد محمود    | 85        | 78     | 65        | 80     | 28/30  |
| سارة علي      | 90        | 88     | 92        | 85     | 30/30  |
| محمود حسن     | 65        | 70     | 68        | 72     | 25/30  |
```

### 2. 📝 نظام الامتحانات الذكي

**إنشاء امتحانات تلقائيًا بالـ AI:**

المدرس يدخل:
- 📚 المادة (مثلاً: رياضيات)
- 📖 الموضوع (مثلاً: المعادلات من الدرجة الثانية)
- ⚙️ مستوى الصعوبة (سهل / متوسط / صعب)
- 🔢 عدد الأسئلة المطلوب

النظام ينشئ تلقائيًا:
- ✅ أسئلة اختيار من متعدد (MCQ)
- ✅ أسئلة صح وخطأ (True/False)
- ✅ أسئلة مقالية (Essay Questions)
- ✅ **نموذج الإجابة** (Answer Key)

**مثال:**
```
المادة: رياضيات
الموضوع: المعادلات من الدرجة الثانية
الصعوبة: متوسط
عدد الأسئلة: 10

النتيجة:
┌────────────────────────────────────┐
│  🎯 امتحان رياضيات - المعادلات  │
├────────────────────────────────────┤
│  1. ما هو حل المعادلة x² - 5x + 6 = 0؟
│     A) x = 2, x = 3 ✅
│     B) x = 1, x = 6
│     C) x = -2, x = -3
│     D) لا يوجد حل
│
│  2. صح أم خطأ: جذور المعادلة...
│  ...
│  10. اشرح كيفية حل المعادلة...
└────────────────────────────────────┘

📄 نموذج الإجابة مرفق تلقائيًا
```

**رفع امتحانات يدوية:**
- المدرس يرفع ملف PDF أو صورة
- النظام يخزنها ويعرضها للطلاب

### 3. 📚 مولد محتوى الدروس بالذكاء الاصطناعي

**كيف يعمل:**

المدرس يدخل:
- 📝 عنوان الدرس (مثلاً: "دورة الماء في الطبيعة")
- 💡 فكرة أو نقاط رئيسية

النظام (AI) ينتج:
- 📖 **شرح مبسط وسلس** للدرس
- 🗂️ **تنظيم منهجي**: مقدمة / شرح تفصيلي / أمثلة / ملخص
- 🖼️ **صور توضيحية** تعليمية مناسبة (بالذكاء الاصطناعي)
- 📄 **ملف PDF** للطباعة
- 📝 **ملف Word** للتعديل

**مثال:**

```
المدخل:
- العنوان: دورة الماء في الطبيعة

│  3. الهطول: تسقط الأمطار...         │
│                                      │
│  🖼️ [صورة توضيحية لدورة الماء]    │
```

### 4. 📖 المكتبة الرقمية

**رفع المحتوى:**
- الكتب المدرسية (PDF)
- الملازم والمذكرات
- شروحات إضافية
- فيديوهات تعليمية (روابط)
- **مصادر خارجية** مفيدة للطالب

**التحميل:**
- الطلاب يحملون المحتوى مجانًا
- التنظيم حسب المادة والفصل
- بحث سريع في المكتبة

**مثال:**
```
📚 المكتبة الرقمية - الصف الثالث الإعدادي

  🎥 شرح المعادلات (فيديو - YouTube)

  📄 كتاب الوزارة (PDF)
  📄 مذكرة النحو (PDF)
  📄 قصص قصيرة للقراءة (PDF)

📁 العلوم
  📄 كتاب الوزارة (PDF)
  📄 تجارب معملية (PDF)
  🎥 تجربة الكهرباء (فيديو)
```

### 5. ✅ نظام الواجبات الذكي

**إنشاء الواجب:**
- المدرس ينشئ واجب بعد كل درس
- يحتوي على **5 أسئلة** (مثلاً)
- أسئلة اختيار من متعدد أو صح وخطأ

**الحل والتصحيح:**
- الطالب يحل الواجب أونلاين
- **التصحيح تلقائي** فورًا
- الطالب يعرف درجته مباشرة

**التتبع:**
- المدرس يتابع من حل ومن لم يحل
- إحصائيات عن أداء الطلاب في الواجبات

**مثال:**
```
📝 واجب رياضيات - درس المعادلات

1. حل المعادلة: 2x + 5 = 13
   A) x = 3
   B) x = 4 ✅
   C) x = 5
   D) x = 6

2. صح أم خطأ: 3 + 5 = 8 ✅

...
```

### 1️⃣ التقييم الدراسي (Academic Evaluation)

التقييم الرسمي المبني على رصد المدرس داخل المدرسة.

**مكونات التقييم:**

| المكوّن | الوصف |
|---------|--------|
| 📝 أعمال السنة الأسبوعية | يرصدها المدرس أسبوعيًا (مثل: أداء منزلي، تقييم أسبوعي، سلوك ومواظبة) |
| 📊 متوسطات الشهور | يُحسب متوسط كل شهر من مجموع الأسابيع |
| 📝 الاختبارات الشهرية | اختبار شهري أول + اختبار شهري ثانٍ |
| 📄 مجموع الأعمال التحريرية | متوسط الشهور + الاختبارات الشهرية (من 70 درجة) |
| 🎓 امتحان آخر العام | درجة الامتحان النهائي (من 30 درجة) |
| 🏆 المجموع النهائي | الأعمال التحريرية + آخر العام = 100 درجة |

**هيكل قاعدة البيانات للتقييم الدراسي:**

```
templates   → قالب السجل (المادة / الصف / معايير التقييم / توزيع الأسابيع على الشهور)
classes     → الفصل الدراسي (مرتبط بقالب + مدرس + مادة + سنة)
students    → الطلاب (مرتبطون بالفصل)
grades      → درجات أسبوعية (values_json لكل معيار + إجمالي أسبوعي)
absence     → غياب أسبوعي (days_json لكل يوم + إجمالي الغياب)
monthly_averages → متوسط كل شهر لكل طالب
monthly_exams    → درجات الاختبارات الشهرية (exam1 + exam2)
final_grades     → الدرجات النهائية (months_avg + written_total + final_exam + total)
```

**مرونة القوالب (Templates):**

النظام يدعم قوالب مخصصة لكل مرحلة دراسية ومادة:
- **عدد الأسابيع**: مرن (14 أسبوع، 18 أسبوع، إلخ)
- **معايير التقييم الأسبوعي**: قابلة للتخصيص بالاسم والدرجة القصوى
  - مثال ابتدائي: حفظ (10) + فهم (10) + تطبيق (10) + نشاط (10)
  - مثال إعدادي: أداء منزلي (10) + تقييم أسبوعي (20) + سلوك ومواظبة (10)
- **توزيع الشهور**: كل شهر يضم مجموعة أسابيع محددة
- **عمود الملخص النهائي**: قابل للتخصيص الكامل

**التحكم في إظهار النتائج:**

```
الإدارة لديها صلاحية كاملة:
✅ إظهار النتائج للطلاب وأولياء الأمور
🔒 إخفاء النتائج (قبل موعد الإعلان)
⏰ فتح النتائج لفترة محددة

💡 فكرة تجارية: يمكن ربط ظهور النتيجة بموافقة الإدارة
   أو جعله مشروطًا بسداد المصروفات أو أي إجراء إداري
```

### 2️⃣ التقييم التدريبي (Training Evaluation)

تقييم ذكي مبني على تفاعل الطالب مع منصة AI School نفسها — يقيس المستوى الحقيقي بشكل مستمر.

**يعتمد على:**
- ✅ الواجبات الإلكترونية وتسليمها
- 📝 التدريبات والتمارين داخل المنصة
- 🎯 الامتحانات الإلكترونية (MCQ / صح وخطأ / أكمل / مقالي)
- 💬 التفاعل مع المحتوى التعليمي
- 📊 نسبة إتمام الدروس

**الهدف:** قياس تطور الطالب الحقيقي بصورة مستقلة عن رصد المدرس.
